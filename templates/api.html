Our APIs

POST

https://password-strength-classify.herokuapp.com/api?password=[password]

A POST request that has 1 parameter, the password and it returns a JSON response containing the strength of entered password.


GET

https://password-strength-classify.herokuapp.com/api/[password]

A GET request that has 1 parameter, the password and it returns a JSON response containing the strength of entered password.
